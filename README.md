# Лабораторная работа №5: Многофайловые проекты. STL

Выполнила: Гольсман Алина Эдуардовна, студент группы АБ-522, АВТФ
Дисциплина: Программирование
Университет: НГТУ (НЭТИ)

## Описание
Данная лабораторная работа посвящена изучению структуры многофайловых проектов на языке C++ и освоению стандартной библиотеки шаблонов (STL). 

Основные цели работы:
* Проектирование многофайловых программ с разделением на заголовочные файлы (`.h`) и файлы реализации (`.cpp`).
* Использование STL-контейнеров (`vector`, `map`, `stack`, `set`) для обработки данных.
* Реализация логики управления ресурсами, комбинации контейнеров и циклической обработки событий.

## Содержание заданий
Работа включает реализацию четырех функциональных задач:

1. **Учет товаров на складе**: управление ячейками хранения, добавление и удаление товаров, мониторинг загруженности.
2. **Управление контейнерами в порту**: моделирование LIFO-стеков и распределение контейнеров по секциям судна.
3. **График движения самолетов**: создание системы связей «самолет — город» с использованием перечислений (`enum class`).
4. **Автоматизация учебного расписания**: работа с календарем, сохранение расписания при переходе на следующий месяц.

## Структура проекта
Проект организован по принципу раздельной компиляции:
* `*.h` — заголовочные файлы с объявлениями структур и прототипов функций.
* `*.cpp` — файлы с реализацией функций.
* `main.cpp` — точка входа в программу, содержащая цикл обработки пользовательского ввода.
